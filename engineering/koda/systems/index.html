How to read this catalogue

IMPLEMENTED, VALIDATED, OBSERVED, DESIGNED, PLANNED, EXPERIMENTAL and HISTORICAL are evidence classes used in the pages. The visible state is a concise current classification. A page may describe an implemented component, an experimental result and planned work without treating them as equivalent.

Scope boundary: This is architecture documentation, not a claim that Koda is a completed autonomous development environment, a complete desktop shell, or a deployed model platform.
