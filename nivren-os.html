Canonical statement

In one sentence

Nivren OS is a local-first AI operating system developed by YewForge.

Expanded: Nivren OS is a genuine Linux-based AI operating system being built to coordinate specialist models, memory, applications, system services and the computer itself.

01 What Nivren is A local-first AI operating system — not SaaS, not a productivity dashboard, not a browser-based OS, not a subscription chatbot layer. It is being built to live on the machine.
02 What makes it an operating system Desktop and interfaces, system services, model orchestration, memory, tools and the ability to coordinate the computer itself — beyond a single chat window.
03 Local-first architecture Designed to run on ordinary local hardware with inspectable state, human authority and privacy-aware boundaries. Cloud convenience is not the product identity.
04 One identity, many specialist models Different models and non-AI tools are treated as bounded contributors under one governed operating environment, rather than one unquestioned authority.
05 Nivren Chat Conversational interfaces are part of the system surface — not the whole product. Chat is a doorway into the operating environment, not a substitute for it.
06 Nivren Core Core runtime, services and coordination layers that hold identity, memory pathways, tools and system state together. Under active engineering; not claimed as finished.

Status honesty

Working / active development / roadmap

In active development. Deeper model orchestration, memory continuity, organ/runtime services, desktop cohesion and clearer human-governed control surfaces.

Roadmap. Broader integrated operating-environment behaviour, wider specialist coverage and future surfaces. Roadmap items are not claimed as present capability.

Honest history

Working name

Nivren was previously developed under the working name Koda. Historical articles, engineering paths and older screenshots may still say Koda where that was accurate at the time. Active public branding now uses Nivren / Nivren OS.
